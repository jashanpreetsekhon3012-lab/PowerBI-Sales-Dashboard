# PowerBI-Sales-Dashboard
📌 Project Overview
This project presents an interactive Sales Dashboard developed using Power BI to analyze sales performance across different regions, product categories, payment methods, and customer types. The project also includes a promotional Canva Poster and Zapier automation (Zaps) to demonstrate workflow automation.

The dashboard helps users monitor sales metrics and make data-driven business decisions through interactive visualizations.

🚀 Features
* Interactive Power BI Dashboard
* Sales performance analysis
* Region-wise sales insights
* Product category comparison
* Payment method analysis
* Customer type distribution
* Dynamic slicers for filtering
* Canva promotional poster
* Zapier workflow automation

🛠️ Tools & Technologies
* Power BI Desktop
* Microsoft Excel (Dataset)
* Canva
* Zapier
* GitHub

📈 Dashboard Highlights
* Sales overview using KPI cards
* Product Category Analysis
* Region-wise Sales Distribution
* Payment Method Analysis
* Customer Type Analysis
* Interactive Filters (Slicers)
* Geographic Sales Visualization
